# Project 4 – Relational Database and SQL JOINs

## Overview
This project demonstrates relational database design using MySQL. It includes table creation, Primary Keys, Foreign Keys, and SQL JOIN queries to analyze relationships between customers, products, and orders.

## Business Questions
1. Which customer purchased which product?
2. What products have been ordered?
3. What is the total quantity sold by product?
4. Which customers placed the most orders?

## SQL Skills Used
- CREATE TABLE
- PRIMARY KEY
- FOREIGN KEY
- INNER JOIN
- GROUP BY
- ORDER BY
- SUM()
