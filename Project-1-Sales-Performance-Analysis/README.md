# Project 1 – Sales Performance Analysis

## Overview
This project analyzes sales performance using MySQL.

## Business Questions
1. What are the total sales and total profit?
2. Which regions generate the highest sales?
3. Which product categories are the most profitable?
4. Who are the top customers?
5. Which products generate the highest revenue?

## SQL Skills Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- SUM()
- COUNT()
- LIMIT

## Tools
- MySQL
- MySQL Workbench
