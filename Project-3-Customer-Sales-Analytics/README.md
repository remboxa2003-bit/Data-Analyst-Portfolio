# Project 3 – Customer Sales Analytics

## Overview
This project analyzes customer purchasing behavior using MySQL. The analysis identifies top customers, sales performance, and customer spending patterns to support business decision-making.

## Business Questions
1. Which customers generate the highest revenue?
2. What is the total sales by customer?
3. Which customers are the most profitable?
4. How does customer purchasing behavior vary?
